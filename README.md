# MediMart - Online Pharmacy App

## Quick Setup
1. Extract: unzip MediMart-app.zip && cd MediMart  
2. Install: npm install  
3. Start: npx expo start -c  
4. Scan QR with Expo Go  

## Features (Like Arogga)
- Medicine search by name, generic, company
- Category browsing with sort and filter
- Medicine detail with drug info and alternatives
- Upload prescription via camera or gallery
- Cart and checkout with delivery calculation
- Order placement and 4-step tracking
- Bilingual English and Bangla
- 20 real BD medicines in local database
- Discount badges and Rx indicators

## Connect to Backend
Edit src/config/api.js with your WooCommerce API keys for medimart.com.bd
